# 게시판 서비스 구현

기본적이고 보편적인 게시판 기능을 포함한 서비스입니다. 

수업과 관련된 내용을 경험할 수 있도록 구현되었습니다.

## 개발 환경

* Intellij IDEA Community
* Java 17
* Gradle 8.1.1
* Spring Boot 3.0

## 기술 세부 스택

Spring Boot
* Spring Web
* Spring Data JPA
* Thymeleaf
* Spring Security
* OAuth2Client
* H2 Database
* MySQL Driver
* Lombok
